# QuantStream API Endpoints

Base URL: `http://localhost:8085/api/v1`

## Market Data Endpoints

### GET /tokens
Get all tokens with latest prices.

**Response:**
```json
{
  "tokens": [
    {
      "symbol": "AAPL",
      "current_price": 175.53,
      "volume": 50000000,
      "last_updated": "2026-07-20T17:08:28.387155Z"
    }
  ],
  "total": 10,
  "last_update": "2026-07-20T17:08:30Z"
}
```

### GET /tokens/{symbol}
Get detailed 24h statistics for a token.

**Example:** `/tokens/AAPL`

**Response:**
```json
{
  "symbol": "AAPL",
  "current_price": 175.53,
  "change_24h_pct": 0.08,
  "high_24h": 810.75,
  "low_24h": 175.37,
  "volume_24h": 1119243440221,
  "open_24h": 175.38,
  "last_updated": "2026-07-20T17:08:28.387155"
}
```

### GET /tokens/{symbol}/candles
Get historical OHLC candles.

**Query Parameters:**
- `interval`: Candle interval (only "1m" supported)
- `start_date`: Start date ISO format (optional)
- `end_date`: End date ISO format (optional)
- `limit`: Max candles (1-5000, default 1000)

**Example:** `/tokens/AAPL/candles?limit=100`

**Response:**
```json
{
  "symbol": "AAPL",
  "interval": "1m",
  "candles": [
    {
      "timestamp": "2026-07-20T14:23:50Z",
      "open": 564.0,
      "high": 572.16,
      "low": 560.62,
      "close": 571.27,
      "volume": 41922134
    }
  ],
  "count": 100
}
```

### GET /tokens/{symbol}/tick
Get most recent tick (price update).

**Example:** `/tokens/AAPL/tick`

**Response:**
```json
{
  "symbol": "AAPL",
  "price": 175.53,
  "volume": 1000,
  "timestamp": "2026-07-20T17:08:28.387155Z"
}
```

**Response:**

```json
    {

      "parameters": {"rsi_period": 14, "oversold": 30, "overbought": 70},
      "active": true

  "total": 10

```

### GET /strategies/{name}

### GET /strategies/leaderboard
Get strategy performance leaderboard (PLACEHOLDER - returns empty).

**Query Parameters:**
- `sort_by`: Field to sort by (default: total_return_pct)
- `period`: Time period (all, 30d, 7d, 24h)

**Response:**
```json
{
  "leaderboard": [],
  "period": "all",
  "sorted_by": "total_return_pct",
  "total_strategies": 10
}
```

### GET /strategies/{name}/signals
Get signals generated by a strategy.

**Query Parameters:**
- `symbol`: Filter by symbol (optional)
- `start_date`: Start date ISO format (optional)
- `end_date`: End date ISO format (optional)
- `limit`: Max signals (1-500, default 100)

**Example:** `/strategies/rsi_strategy/signals?symbol=AAPL&limit=50`

**Response:**
```json
{
  "signals": [
    {
      "timestamp": "2026-07-20T11:15:30Z",
      "symbol": "AAPL",
      "strategy_name": "rsi_strategy",
      "action": "BUY",
      "price": 0.0,
      "confidence": 0.85
    }
  ],
  "count": 50,
  "last_updated": "2026-07-20T11:15:35Z"
}
```

## Signal Endpoints

### GET /signals/recent
Get recent trading signals across all strategies.

**Query Parameters:**
- `limit`: Max signals (1-500, default 50)
- `action`: Filter by action (BUY, SELL, HOLD)
- `symbol`: Filter by symbol

**Example:** `/signals/recent?limit=20&action=BUY`

**Response:**
```json
{
  "signals": [
    {
      "timestamp": "2026-07-20T11:15:30Z",
      "symbol": "AAPL",
      "strategy_name": "rsi_strategy",
      "action": "BUY",
      "price": 0.0,
      "confidence": 0.85
    }
  ],
  "count": 20,
  "last_updated": "2026-07-20T11:15:35Z"
}
```

### GET /signals/symbol/{symbol}
Get all recent signals for a specific symbol.

**Query Parameters:**
- `limit`: Max signals (1-500, default 100)

**Example:** `/signals/symbol/AAPL?limit=50`

### GET /signals/statistics
Get aggregate signal statistics.

**Query Parameters:**
- `period_hours`: Time period in hours (1-168, default 24)

**Example:** `/signals/statistics?period_hours=24`

**Response:**
```json
{
  "statistics": {
    "period_hours": 24,
    "total_signals": 1250,
    "by_action": {
      "BUY": 450,
      "SELL": 380,
      "HOLD": 420
    },
    "by_strategy": {
      "rsi_strategy": 300,
      "momentum_strategy": 280
    },
    "by_symbol": {
      "AAPL": 125,
      "GOOGL": 118
    },
    "avg_confidence": 0.7234
  }
}
```

---

## Frontend Polling Strategy

For real-time updates without WebSocket:

1. **Market Prices**: Poll `/tokens` every 2-3 seconds
2. **Signal Feed**: Poll `/signals/recent` every 5 seconds
3. **Strategy Stats**: Poll `/strategies/{name}` every 10 seconds
4. **Health Check**: Poll `/health` every 30 seconds

## Notes

- All timestamps are in ISO 8601 format (UTC)
- Price field in signals is 0.0 (not stored in signals table, can be fetched from ticks if needed)
- Leaderboard and performance endpoints return placeholder data (future: compute from backtest_results)
- All endpoints use polling (no WebSocket/SSE)
